Instructions for installing the Ilex compiler will be available here once the compiler reaches its first public release.

## Building from Source

For now, you can build Ilex from source:

```bash
git clone https://github.com/ilex-lang/ilex.git
cd ilex
make
```

## Next Steps

Once installed, follow the [Hello World](/getting-started/hello-world/) tutorial to write your first Ilex program.
